## Car Sales Prediction using Machine Learning
### Project Overview

This project focuses on predicting car sales using machine learning techniques by analyzing factors such as advertising spend, audience segmentation, and platform performance to help businesses optimize their marketing strategies and maximize revenue.

### Objectives
* Analyze the relationship between advertising and sales
* Perform data cleaning and exploratory data analysis (EDA)
* Build and evaluate machine learning models for sales prediction
* Identify key factors influencing car sales
### Dataset Description

The dataset contains information related to:

* Advertising expenditure (TV, Radio, Newspaper, etc.)
* Target audience characteristics
* Sales figures

These features are used to train a predictive model for estimating future sales.

### Technologies Used
* Python
* Jupyter Notebook
### Project Workflow
1. Data Collection
Imported dataset for analysis
2. Data Preprocessing
Handled missing values
Checked data types
Cleaned and structured the dataset
3. Exploratory Data Analysis (EDA)
Visualized relationships between features and sales
Used correlation matrix and scatter plots
Identified key trends and patterns
4. Feature Engineering
Selected relevant features for model building
5. Model Building
Applied machine learning algorithms such as:
Linear Regression
Trained model on dataset
6. Model Evaluation
Evaluated performance using metrics like:
R² Score
Mean Squared Error (MSE)
### Results & Insights
* Advertising spend shows a strong influence on sales
* Certain platforms contribute more effectively to revenue generation
The model provides reliable predictions for future sales trends
