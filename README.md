# Movie Rating Prediction using Machine Learning
### Project Overview

This project aims to predict movie ratings using machine learning techniques by analyzing features such as genre, audience preferences, and other relevant attributes. The goal is to build a model that can estimate ratings accurately and help understand factors influencing movie success.

### Objectives
* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Identify key features affecting movie ratings
* Build and evaluate a machine learning model for prediction
### Dataset Description

The dataset includes information such as:

* Movie title
* Genre
* Year of release
* Ratings
* Additional relevant features

These variables are used to train a predictive model.

### Project Workflow
1. Data Collection
Loaded dataset for analysis
2. Data Preprocessing
Handled missing values
Removed duplicates
Converted data types
3. Exploratory Data Analysis (EDA)
Visualized rating distributions
Analyzed trends across genres and years
Used correlation heatmaps and plots
4. Feature Engineering
Selected important features
Encoded categorical variables
5. Model Building
Applied machine learning algorithms such as:
Linear Regression / other regression models
6. Model Evaluation
* Evaluated using:
a) Mean Squared Error (MSE)
b) R² Score
### Results & Insights
* Certain genres and features significantly influence ratings
* The model can reasonably predict movie ratings
* Data patterns help understand audience preferences
