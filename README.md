### 1ST PROJECT FOR CODSOFT INTERNSHIP
# CREDIT CARD FRAUD DETECTION PROJECT
* DATASET URL: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?select=creditcard.csv
* #### PROJECT OVERVIEW
* The project focuses on building a robust machine learning pipeline to detect fraudulent credit card transactions.
* Given the highly IMBALANCED DATASET, the goal is to accurately identify fraud cases while minimizing false positives.

* #### KEY FEATURES
*  End-to-end ML pipeline (EDA → Preprocessing → Modeling → Evaluation)
*  Handling class imbalance using SMOTE
*  Multiple model comparison:
        Logistic Regression,
        Polynomial Logistic Regression,
        Random Forest,
        XGBoost

#### RESULTS AND INSIGHTS
Key Insight:
In fraud detection, recall is more important than accuracy, as missing a fraud is more costly than a false alarm.
